# Instructions.md · сайт «Бугаков Электромонтаж»

## 1. Цель проекта

Создать современный одностраничный сайт-визитку для частного мастера по электромонтажным работам в Омске.

Главная цель сайта:
- быстро объяснить, какие электромонтажные работы выполняются;
- вызвать доверие к мастеру;
- показать фото работ и блок «до/после»;
- привести посетителя к звонку по телефону;
- дать возможность быстро написать в Telegram или Max;
- быть удобным на телефоне.

Сайт должен быть простым в использовании, визуально сильным, аккуратным, не перегруженным, с тёмным профессиональным дизайном и эффектом электрической схемы с бегущими импульсами.

## 10. Фото работ

Нужна галерея работ.

Требования:
- сделать блок «Фото работ»;
- фото должны быть крупными, но не перегружать страницу;
- на мобильной версии фото должны открываться удобно;
- предусмотреть простое добавление новых фото через папку.

Папка для фото:

`images/works/`

Желательно сделать отдельный файл с данными галереи:

`data/gallery.json`

Пример структуры:

```json
[
  {
    "src": "images/works/work-01.jpg",
    "title": "Сборка электрощита",
    "category": "Щитки"
  },
  {
    "src": "images/works/work-02.jpg",
    "title": "Разводка электрики",
    "category": "Проводка"
  }
]
```

Если `gallery.json` не использовать, можно сделать массив в JS-файле. Но лучше вынести в отдельный файл, чтобы пользователь мог добавлять фото без правки HTML.

## 11. До / после

Нужен отдельный блок «До / после».

Требования:
- блок должен показывать пары фото;
- можно сделать карточки: слева «до», справа «после»;
- для мобильной версии — вертикальное расположение;
- предусмотреть возможность самостоятельного добавления фото.

Папка:

`images/before-after/`

Желательно файл:

`data/before-after.json`

Пример структуры:

```json
[
  {
    "before": "images/before-after/before-01.jpg",
    "after": "images/before-after/after-01.jpg",
    "title": "Замена электрощита"
  }
]
```

## 12. Как проходит работа

Важно: по телефону мастер даёт предварительный ориентир, а после выезда делает более точную оценку.

Структура блока:

1. Звонок или сообщение
2. Клиент описывает задачу
3. Мастер предварительно ориентирует по цене и срокам
4. Выезд на объект
5. Более точная оценка стоимости после осмотра
6. Выполнение работ
7. Проверка результата
8. Гарантия на выполненные работы

Формулировка:

> Сначала мастер уточняет задачу по телефону и ориентирует по стоимости. После выезда и осмотра даёт более точную оценку, согласует объём работ и только после этого приступает к выполнению.

## 13. FAQ

Добавить блок частых вопросов.

Вопросы и ответы:

### Берёте мелкие заявки?
Да, берём. Минимальная сумма работ — от 2 000 ₽.

### Можно ли вызвать мастера только на осмотр?
Да. Если работы не выполняются, осмотр/оценка — 500 ₽. Если работы выполняются, выезд входит в стоимость.

### Работаете по пригородам?
Да, по договорённости. Удалённые объекты — от 1 000 ₽ за выезд, итоговая стоимость обсуждается индивидуально.

### Кто покупает материалы?
Можно по-разному: клиент покупает сам, мастер помогает подобрать список или покупает материалы по согласованию.

### Есть ли гарантия?
Да, на выполненные работы предоставляется гарантия.

### Можно ли срочно?
По возможности. Если мастер свободен, можно договориться на ближайшее время.

### Работаете с офисами и магазинами?
Да, с небольшими коммерческими помещениями условия обсуждаются индивидуально.

## 16. Анимации и вау-эффект

Нужна мягкая анимация при ручной прокрутке страницы.

Анимации:
- блоки плавно появляются при скролле;
- карточки услуг поднимаются снизу;
- прайс проявляется мягко;
- фото работ появляются с лёгким увеличением;
- кнопки подсвечиваются при наведении;
- всё должно быть плавно, без резких скачков.

Главный вау-эффект:

**Схема проводки с импульсами**

Описание:
- на фоне или между блоками идут тонкие линии, похожие на электрическую схему/проводку;
- при прокрутке по линиям пробегают световые импульсы;
- импульсы жёлто-оранжевые;
- эффект должен выглядеть профессионально и аккуратно;
- не делать большие молнии на весь экран;
- не перегружать страницу;
- на мобильных устройствах использовать облегчённую версию.

Формулировка для реализации:

> Добавить аккуратный визуальный эффект электрической схемы: тонкие линии как проводка, по которым при прокрутке пробегают световые импульсы. Эффект должен быть фоновым, профессиональным, не мультяшным и не мешать чтению.
